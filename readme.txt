Welcome to the initial prototype of BeAmI.io.

Basic functionality has been implemented.

To access:

1.) Download source code files from github repository at --->    https://github.com/syivan/BeAmI.git  <-----

2.) Enter a terminal

3.) Access the directory in the terminal where the source code files were installed

4.) Enter "node app.js"

5.) Go to your browser and enter the local address of the prototype: "localhost:3000"

6.) Test out the functionality


Using the Web App:

1.) The Web application uses the input given from the user and calculates the necessary calculations to return to the user fitness information
that they want to retrieve. The user may simply go to a tab and enter the information necessary to calculate things such as Body Mass Index, Ideal Weight Suggested, etc.
